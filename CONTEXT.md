# Project Context & Rules

This document outlines core instructions, constraints, and guidelines for development within the Wadiolk repository.

## Formatting & System Output Rules

### Emoji Policy
- **DO NOT** use any emojis (表情符號) in terminal outputs, error traces, warnings, success banners, comments, or commit messages.
- Use uppercase tags (e.g., `[ERROR]`, `[WARN]`, `[SUCCESS]`, `[INFO]`) along with ANSI colors to display status levels instead of visual emojis. Keep terminal and file environments entirely clean, professional, and text-only.
